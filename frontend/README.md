## 🎯 Что здесь происходит?

Фронтенд CarMatch — пользовательский интерфейс, через который ты проходишь регистрацию, отвечаешь на вопросы, смотришь результаты и плачешь от счастья (или разочарования). Здесь есть анимации, красивые градиенты и вообще всё, чтобы ты забыл, что внутри просто 5 вопросов.

## 🧱 Что внутри?

| Компонент | Зачем |
|-----------|-------|
| React 18 | Главный строитель страниц |
| TypeScript | Чтобы ошибки ловились до того, как ты увидишь белый экран |
| Vite | Очень быстрая сборка (ни секунды ожидания) |
| Axios | Стучится в бэкенд, пока ты кликаешь |
| React Router | Переключает страницы без перезагрузки |
| Framer Motion | Анимации, от которых тает сердечко |

## 🏃 Запуск (всё просто, честно)

```bash
# Ставим зависимости (это долго, сходи за чаем)

# Запускаем разработку (магия произойдёт на localhost:5173)

# Собираем для публикации (минификация, сжатие, всё серьёзно)

# Смотрим, что получилось (продакшен версия локально)
npm run preview

🎨 Страницы (какие есть и за что отвечают)

/ — Landing
Красивая реклама с карточками преимуществ, кнопки «Войти» и «Регистрация». Если уже залогинен — перекидывает в /home.

/login — Вход
Поле для username, поле для пароля, кнопка входа. Если ошибся — красная надпись и чувство вины.

/register — Регистрация
Имя, почта, пароль (и подтверждение, чтобы ты точно помнил). После успеха — автоматический вход.

/home — Дашборд
Форма из 5 вопросов. Выбирай варианты, жми «Подобрать». После клика — анимация загрузки и переход на результат.

/result — Результат
Большая карточка с маркой авто, эмодзи и обоснованием выбора. Оттуда можно в историю или новый подбор.

/history — История
Список всех твоих подборов. Можно удалить неудачный (подбор, не автомобиль). Сортировка от новых к старым.
