## 🎯 Что это вообще такое?

**CarMatch** — это веб-приложение, которое подбирает тебе автомобиль. Не по скучным техническим характеристикам, а по твоему вайбу: какой у тебя любимый цвет, пьёшь ты кофе или чай, носишь кэжуал или костюм.

Результат выходит с эмодзи и пафосным обоснованием. Можно сохранять историю, пересматривать старые подборы и удалять неудачные (как неудачные отношения).

## 🧠 Как это работает?

1. Ты регистрируешься (или заходишь, если уже был)
2. Отвечаешь на 5 вопросов
3. Бэкенд колдует над ответами
4. Получаешь рекомендацию с крутой анимацией
5. Сохраняешь результат в историю


## 🏗️ Технологии (почему это не упало при первом запуске)

| Часть | Технологии |
|-------|-------------|
| Фронт | React, TypeScript, Vite, Axios |
| Бэк | FastAPI, Python, SQLAlchemy |
| БД | PostgreSQL |
| Анимации | Framer Motion (потому что красиво) |
| Авторизация | JWT + bcrypt (пароли не в открытую, не волнуйся) |

## 🚀 Быстрый старт (для нетерпеливых)

```bash
# Запускаем бэк
cd backend
python -m venv venv && source venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload

# В новом терминале запускаем фронт
cd frontend
npm install
npm run dev

# Открывай браузер на http://localhost:5173

```
